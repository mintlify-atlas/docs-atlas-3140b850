<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with .NET Aspire in minutes
  </p>

  <Steps>
    <Step title="Install the Aspire CLI">
      Install the Aspire CLI using the installation script for your platform:

      <CodeGroup>
        ```bash Windows (PowerShell)
        irm https://aspire.dev/install.ps1 | iex
        ```

        ```bash Linux/macOS
        curl -sSL https://aspire.dev/install.sh | bash
        ```
      </CodeGroup>

      The CLI provides everything you need to create, run, and deploy Aspire applications.
    </Step>

    <Step title="Create a new Aspire app">
      Create your first Aspire application using the starter template:

      ```bash
      aspire init my-aspire-app
      cd my-aspire-app
      ```

      This creates a new solution with an AppHost project and a starter web application, fully configured with service defaults.
    </Step>

    <Step title="Run your application">
      Launch your application with a single command:

      ```bash
      aspire run
      ```

      The Aspire dashboard opens automatically, showing your running services, logs, traces, and metrics in real-time.

      <Note>
        The dashboard runs at `http://localhost:18888` by default and provides live telemetry from all your services.
      </Note>
    </Step>
  </Steps>
</div>
